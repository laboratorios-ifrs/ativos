# 🖥️ Gestão de Laboratórios e Mapa de Ativos

Sistema web desenvolvido com Google Apps Script (HTML, CSS, JavaScript) e Google Sheets como banco de dados. Permite o mapeamento visual de computadores, pontos de rede e estruturas físicas, além de gerenciar fichas técnicas completas de hardware.

🔗 **[Acessar o Sistema em Funcionamento (Live Demo)](https://script.google.com/macros/s/AKfycbwwFx7R88bT5fTq9vIb_tYZIja0eQhgDjWqEXXvaqMTUrQDuStvcB6i5dOhTnkWiK_A/exec)**

## 🚀 Funcionalidades
* Mapa interativo com visualização de Gabinetes, Monitores e Rede (Drag and Drop).
* Banco de dados integrado via Google Sheets (21 colunas de ficha técnica).
* Busca rápida global por Patrimônio ou Service TAG.
* Sistema de login e controle de transferência de ativos entre laboratórios.

## 🛠️ Tecnologias Utilizadas
* Front-end: HTML5, CSS3, Vanilla JavaScript
* Back-end: Google Apps Script (`codigo.gs`)
* Banco de Dados: Google Sheets
